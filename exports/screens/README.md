# Mowa — статические экраны

Набор автономных HTML-макетов для просмотра, согласования и экспорта. Начните с `index.html` — на нём есть ссылки на все экраны.

## Экспорт

- Desktop: откройте экран при ширине `1440 px`.
- Mobile: используйте viewport `390 × 844 px`.
- PDF: в окне печати включите фоновые изображения и поля «Нет».
- PNG: сделайте полноразмерный снимок страницы средствами браузера.

Все экраны используют общий `styles.css`, не требуют сборки, сервера, JavaScript или доступа к интернету.
